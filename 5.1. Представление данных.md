Данные в формате с плавающей точкой
Данные в формате с плавающей точкой. Формат включает три поля: Знак (S), Порядок и Мантисса. Поле мантиссы содержит значащие биты числа, а поле порядка содержит степень 2 и определяет масштабирующий множитель для мантиссы. Форматы данных поддерживаются блоком обработки чисел с плавающей точкой (FPU).

•Для внутреннего представления и обработки данных используется двоичная система счисления.

•Для представления беззнаковых чисел используются все разряды кода.

•При представлении числа со знаком старший разряд хранит знак числа. Плюс – 0, минус –1.

•Код, в котором для отрицательного числа используется абсолютное значение называется прямым кодом. Например, код числа -3 в виде байта будет 10000011.

•Такой код легко читается человеком, но имеет недостатки для построения логической схемы сумматора. Проблема в первую очередь касается выполнения операции вычитания из-за необходимости реализации заемов.

•Для перехода от операции вычитания к алгебраическому сложению можно использовать обратный код. Код положительного числа в обратном коде эквивалентен коду в прямом коде. Отрицательное число получается из прямого кода инверсией всех цифр прямого кода. Например, -3 в обратном коде будет равно 11111100.
