# **Основные типы данных**

***Основными типами данных в компьютерах интеловской архитектуры являются: байт, слово (2 байта), двойное слово (4 байта) и квадрослово (8 байт).***

Для внутреннего представления и обработки данных ***используется двоичная система счисления.***

Каждый из представленных на скалярных типов данных может начинаться с любого адреса: это означает, что слово не обязано начинаться с чётного адреса; двойное слово – с адреса, кратного 4 и т.д.

# **Целочисленные данные**
Данные с фиксированной точкой ***могут быть как со знаком, так и без знака. Под знак отводится старший бит формата данных***. Для представления беззнаковых чисел используются все разряды кода.

### *Прямой код*
— ***код, в котором для отрицательного числа используется абсолютное значение***. Например, код числа -3 в виде байта будет 10000011.

***Такой код легко читается человеком, но имеет недостатки для построения логической схемы сумматора.*** Проблема в первую очередь касается выполнения операции вычитания из-за необходимости реализации заемов.

### *Обратный код*
— ***код положительного числа в обратном коде эквивалентен коду в прямом коде. Отрицательное число получается из прямого кода инверсией всех цифр прямого кода.*** Например, -3 в обратном коде будет равно 11111100.

***При использовании обратного кода можно перейти от операции вычитания к операции сложения. Но в обратном коде появляется «две версии нуля»*** — положительный ($0000$) и отрицательный ($1111$). ***Это усложняет логику процессора***, поэтому сейчас он почти не используется для хранения чисел. Также ***результатам операций с переносом из старшего разряда требуется коррекция с помощью циклического переноса единицы.***

### *Дополнительный код*
— ***код получается из обратного добавлением 1 к младшему разряду***. Например, дополнительный код числа -3 равен 11111100+1=11111101.

***Необходимость коррекции отпадает***, если использовать для представления отрицательных чисел дополнительный код. 

# **Данные в формате с плавающей точкой**
***Формат включает три поля: Знак (S), Порядок и Мантисса.*** Поле мантиссы содержит значащие биты числа, а поле порядка содержит степень 2 и определяет масштабирующий множитель для мантиссы. ***Поддерживаются блоком обработки чисел с плавающей точкой ([[3.
